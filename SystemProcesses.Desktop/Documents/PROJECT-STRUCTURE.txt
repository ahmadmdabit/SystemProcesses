KEY FEATURES IMPLEMENTED:
-------------------------

1. HIERARCHICAL DISPLAY
   - TreeView with parent-child process relationships
   - Expand/collapse functionality
   - Sorted alphabetically by name

2. COLUMNS (Left to Right):
   - PID (Process ID)
   - Name (with icon, services in grey)
   - CPU (percentage with %)
   - Mem (auto-formatted KB/MB/GB)
   - VM Size (auto-formatted KB/MB/GB)
   - Parameters (command line arguments)

3. FILTERING & SEARCH
   - Real-time search filter (case-insensitive)
   - Tree isolation toggle (show only selected tree)

4. REFRESH CONTROL
   - Configurable rates: 1s, 2s, 5s, 10s
   - Pause/Resume functionality
   - Auto-refresh with DispatcherTimer

5. TOOLBAR ACTIONS
   - End Process (with confirmation)
   - End Process Tree (with confirmation)
   - Show Process Details (dialog)
   - Open Process Directory (Explorer)
   - Isolate Tree (toggle filter)

6. ADVANCED FEATURES
   - Service process detection (using ServiceController)
   - Process icon extraction
   - WMI integration for parent-child relationships
   - Command line parameter retrieval
   - Async/await for non-blocking UI
   - Proper error handling for Access Denied

Method 2 - Visual Studio:
  1. Open SystemProcesses.Desktop.csproj in Visual Studio 2022+
  2. Build > Build Solution (Ctrl+Shift+B)
  3. Debug > Start Debugging (F5)
